🌿 ToVerde - Sistema de Monitoramento Ambiental O ToVerde é uma plataforma full-stack desenvolvida em Django para facilitar a denúncia e o monitoramento de crimes ambientais. O objetivo é conectar cidadãos e autoridades para combater atividades como desmatamento, queimadas e poluição.

Funcionalidades  Autenticação Completa: Sistema de cadastro, login, logout e recuperação de senha por e-mail.

Denúncias Dinâmicas: Submissão de ocorrências com suporte a fotos/vídeos e opção de anonimato.

Painel Administrativo: Interface customizada para gestão de denúncias, com filtros e controle de status.

Perfil do Usuário: Espaço para o usuário logado acompanhar o histórico e o status de suas próprias denúncias.

Design Responsivo: Interface moderna construída com Bootstrap, adaptável para dispositivos móveis.

Tecnologias Utilizadas Linguagem: Python

Framework Web: Django

Frontend: HTML5, CSS3, JavaScript e Bootstrap

Banco de Dados: SQLite (Desenvolvimento)

Versionamento: Git
